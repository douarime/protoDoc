CeRTAE ProtoExt propose une approche basée sur la méthode DATARUN. Cette approche consiste à réaliser les interfaces d’une application par la construction de vues à partir d’un modèle de données standard.

Cette documentation présente un guide d’utilisation aux utilisateurs de cette application.
