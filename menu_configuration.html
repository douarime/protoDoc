Menu des fonctions : configuration¶

Figure 16 : Fonction configuration et son sous-menu.

La fonction Configuration permet l’accès aux fonctions de personnalisation de l’application du prototypeur. Quand vous cliquez sur le bouton Configuration, un sous-menu apparait au-dessous du menu de fonctions. Ce sous-menu est composé des options suivantes: Formulaire, Champs, Détails, Configuration et Méta. Notez que vous pouvez aussi accéder aux options du sous-menu en cliquant sur le petit triangle noir situé à droit du bouton configuration.

Note

L’accès à toutes les options de la fonction Configuration mentionnés dans le paragraphe anterièur est réservé aux utilisateurs avec les droits. Si votre compte n’a pas les droits suffisants, vous verrez uniquement l’option Configuration qui donne accès aux propriétés de configuration de base.
